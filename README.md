# DMG Media Take Home Test
```
A frontend and backend service that communicates with eachother to render Github Organization repository names 
and there respective issues based on given or supplied values.
```

```
 $ Clone the Dmg project then open WebStorm or any IDE of your choice and clone using VCS, Paste the link gotten from github  
 $ Navigate to server folder and peruse the README file
 $ Navigate to client folder and peruse the README file
```
